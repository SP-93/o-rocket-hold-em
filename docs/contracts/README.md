## Deployment:
Deploy se radi ručno iz Admin panela:
1. Poveži admin wallet
2. Idi na /admin
3. Klikni "Deploy Contract"
4. Potpiši transakciju u MetaMask-u
5. Sačekaj confirmation
6. Contract adresa će se automatski sačuvati
