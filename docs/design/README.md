# O'Rocket Hold'em - Design System

## Boje (HSL)

### Primary Colors
- **Primary (Green)**: `142 70% 45%` - Glavni akcent, CTA dugmad
- **Accent (Gold)**: `45 100% 50%` - Pobede, specijalne akcije

### Poker Colors
- **Felt**: `145 60% 22%` - Poker sto pozadina
- **Felt Dark**: `145 65% 15%` - Poker sto outer edge
- **Gold**: `45 100% 50%` - Highlights, pobede
- **Red**: `0 85% 55%` - Destructive, hearts/diamonds

### Chip Colors
- **Blue**: `210 80% 50%`
- **Green**: `142 70% 45%`
- **Black**: `220 20% 15%`
- **White**: `0 0% 95%`

## Fontovi
- **Display**: Orbitron - za naslove, logo, brojeve
- **Body**: Rajdhani - za tekst, dugmad, UI elemente

## Animacije
- `card-deal`: 0.5s - deljenje karata
- `chip-bounce`: 0.5s - čip animacija
- `pulse-glow`: 2s infinite - pulsiranje za aktivne elemente
- `fade-in`: 0.3s - fade in za modals

## Komponente
- Poker sto: oval sa radial gradient
- Karte: 2.5:3.5 aspect ratio
- Čipovi: krug sa highlight

## Layout
- Max width: 1400px
- Mobile first
- Dark mode only
